# PWA Icons

This directory contains the icons for the Progressive Web App.

## Generating Icons

1. Open `icon-generator.html` in your browser (located in the assets folder)
2. Download all the generated icons
3. Save them in this `assets/icons/` directory

## Required Icon Sizes

- 72x72
- 96x96
- 128x128
- 144x144
- 152x152
- 192x192
- 384x384
- 512x512

## Alternative: Use an Online Icon Generator

You can also use online tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

Upload a 512x512 icon and it will generate all the required sizes.
